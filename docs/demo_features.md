# Demo Branch — Early Access Features

The `demo` branch contains early-access and preview features planned for future updates. These are intended for demonstration and testing.

Current planned demo features:

1. Language Creator by C
   - A tool to scaffold new language modules and syntax using C
2. Language Creator by C++
   - C++-based tool providing richer parsing and template helpers
3. Language Creator by C# (requires .NET framework)
   - A .NET-based language creation tool with IDE integrations
4. Visual P++ Redistributable
   - A runtime / UI framework for building visual apps with P++

Note: The `demo` branch is stable for demos but may include features that require external runtimes (e.g., .NET) or additional dependencies. Use the `experimental` branch for experimental tools like `Syntax Creator`.
